# insecure-rds

> ⚠️ **Intentionally insecure.** This module exists to demonstrate Terramantle's
> scan + policy gating. It publishes successfully but is flagged **not
> consumable** until the org's enforcing policies pass (hardcoded credentials,
> public accessibility, no encryption, etc.).

Published to Terramantle as `insecure-rds/aws`. Release a new version by tagging:

```bash
git tag insecure-rds-aws@0.1.0 && git push origin insecure-rds-aws@0.1.0
```

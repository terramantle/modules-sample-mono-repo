# random-pet

Friendly unique name from `random_pet` - a cloud-free demo module. No provider
credentials, applies instantly, perfect for exercising the Terramantle state
backend without a cloud account.

Published to Terramantle as `random-pet/random`. Releases are driven by your
commit messages - merge a Conventional-Commit change to `main` and CI versions,
publishes, and tags it for you:

```bash
git commit -m "feat(random-pet): describe the change"   # minor bump
git commit -m "fix(random-pet): describe the fix"       # patch bump
```

# local-manifest

Renders a templated JSON manifest (with a random release id) to a `local_file`.
Cloud-free, applies instantly. The rendered content also lands in state, giving
the state scanner ordinary non-secret data to walk.

Published to Terramantle as `local-manifest/local`. Releases are driven by your
commit messages - merge a Conventional-Commit change to `main` and CI versions,
publishes, and tags it for you:

```bash
git commit -m "feat(local-manifest): describe the change"   # minor bump
git commit -m "fix(local-manifest): describe the fix"       # patch bump
```

The section below is generated by `terraform-docs` via the pre-commit hook - do
not edit it by hand.

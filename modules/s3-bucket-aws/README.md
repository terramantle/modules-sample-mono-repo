# s3-bucket

S3 bucket with configurable ACL and CORS.

Published to Terramantle as `s3-bucket/aws`. Releases are driven by your
commit messages — merge a Conventional-Commit change to `main` and CI versions,
publishes, and tags it for you:

```bash
git commit -m "feat(s3-bucket): describe the change"   # minor bump
git commit -m "fix(s3-bucket): describe the fix"       # patch bump
```

# http-echo

Fetches an external URL with the `http` data source and echoes the response
into state (mirrors `tf-test/main.tf`). Cloud-free, applies instantly.

This module is **deliberately interesting for the state scanner**: the response
body is stored in state, so any URLs / IPs it contains are visible to
Terramantle's `endpoint_scan`. Pointed at `httpbin.org/ip` by default, the state
ends up carrying your egress IP - exactly the kind of exposed endpoint the
scanner surfaces.

Published to Terramantle as `http-echo/http`. Releases are driven by your commit
messages - merge a Conventional-Commit change to `main` and CI versions,
publishes, and tags it for you:

```bash
git commit -m "feat(http-echo): describe the change"   # minor bump
git commit -m "fix(http-echo): describe the fix"       # patch bump
```
